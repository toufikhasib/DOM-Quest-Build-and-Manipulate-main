What is the Document Object Model (DOM)?

The document object represents your web page. If you want to access any element in an HTML page, you always start with accessing the document object. Below are some examples of how you can use the document object to access and manipulate HTML.

How do you secelet on elements from DOM ?
  • getElementById: Selects an element by its id.
  • getElementsByClassName: Selects elements by class. Returns an HTMLCollection.
  • getElementsByTagName: Selects elements by tag. Returns an HTMLCollection.
  • querySelector: Selects the first element that matches a CSS selector.
What is event delegation in the context of the DOM, and why it is useful ?

Event delegation is a technique where a single event listener on a parent element handles events from its child elements, even if they are added dynamically. It improves efficiency by reducing the number of listeners and simplifies code maintenance.

Hoe do you manipulate an element's attributes and styles using the DOM ?

You can manipulate an element's attributes using getAttribute, setAttribute, and removeAttribute. For styles, use element.style.property to get or set specific styles directly.
